# Architecture Decision Records (ADRs)

This directory captures key architectural decisions using the ADR format.

## Format

Each ADR follows [MADR](https://adr.github.io/madr/) (Markdown Any Decision Records):

```
docs/adr/XXXX-title-of-decision.md
```
